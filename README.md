# Hygiea

A spiritual-hygiene app for practitioners of spiritual science. Forked from Sukoon.

**Stack:** Next.js 16.2.6 · React 19 · TypeScript 5 · Tailwind CSS 4 · Supabase · astronomy-engine

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Project structure

```
src/
  app/          # Next.js App Router pages
  content/      # Authored content (placements, calendar, exercises…)
  i18n/         # Translations — en.ts (default), ar.ts
  lib/          # Core logic (chartCalculator, traitEngine, schema…)
  theme/        # Design tokens (tokens.ts)
```
